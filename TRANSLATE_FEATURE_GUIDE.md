# 翻译功能使用指南

## 功能概述

Threply 键盘现已支持完整的翻译语言选择功能。用户可以在键盘的 AI 面板中选择源语言和目标语言，然后翻译任何文本。

## 如何使用翻译功能

### 步骤 1: 打开 AI 面板

在键盘上输入文本后，点击候选词栏左侧的 **"/"** 按钮打开 AI 技能栏。

### 步骤 2: 切换到翻译模式

在 AI 面板的顶部，你会看到三个模式按钮：
- **B** - 智能回复模式
- **C** - 2D 风格调整模式
- **翻译** - 翻译模式（新增）

点击 **"翻译"** 按钮切换到翻译模式。

### 步骤 3: 选择源语言

在翻译面板中，点击 **"源语言"** 下拉菜单，选择待翻译文本的语言：

| 语言代码 | 语言名称 |
|---------|--------|
| auto | 自动检测 |
| zh | 中文 |
| en | 英文 |
| ja | 日文 |
| ko | 韩文 |
| es | 西班牙文 |
| fr | 法文 |
| de | 德文 |

**提示：** 选择 **"自动检测"** 可让系统自动识别源语言。

### 步骤 4: 选择目标语言

点击 **"目标语言"** 下拉菜单，选择翻译的目标语言。

**注意：** 目标语言不支持 "自动检测"，必须明确选择。

### 步骤 5: 快速交换语言

如果需要交换源语言和目标语言，点击中间的 **"⇄"** 按钮即可快速交换。

**注意：** 如果源语言是 "自动检测"，交换按钮将被禁用。

### 步骤 6: 执行翻译

点击 **"翻译"** 按钮开始翻译。翻译结果会显示在下方的预览区域。

### 步骤 7: 提交翻译结果

翻译完成后，你可以：
- 点击翻译结果将其插入到输入框
- 继续修改源语言或目标语言后重新翻译
- 点击 **"退出"** 按钮关闭 AI 面板

## 翻译来源

系统会自动选择翻译服务：

### 使用 DeepSeek API（推荐）
如果你在设置中配置了 DeepSeek API Key，系统将使用 DeepSeek 的翻译服务。

**优点：**
- 无需后端服务器
- 翻译质量高
- 支持更多语言

**配置方法：**
1. 打开 Threply 主应用
2. 进入设置 → 开发者选项
3. 输入你的 DeepSeek API Key
4. 保存设置

### 使用后端 API（备选）
如果未配置 DeepSeek API Key，系统将使用 Threply 后端服务进行翻译。

**要求：**
- 需要登录 Threply 账户
- 需要网络连接到 Threply 服务器

## 语言选择持久化

你选择的源语言和目标语言会自动保存。下次打开翻译模式时，系统会记住你的选择。

## 常见问题

### Q: 翻译失败，显示错误信息怎么办？

**A:** 可能的原因：
1. 网络连接不稳定 - 检查网络连接
2. 未配置 DeepSeek API Key 且未登录 - 登录或配置 API Key
3. 输入文本为空 - 确保输入了要翻译的文本
4. 后端服务故障 - 稍后重试

### Q: 为什么看不到翻译模式按钮？

**A:** 翻译模式按钮应该在 AI 面板的顶部，与 B 和 C 模式按钮并排显示。如果看不到：
1. 确保已点击 "/" 按钮打开 AI 面板
2. 检查键盘是否已更新到最新版本
3. 尝试重启键盘应用

### Q: 翻译结果不准确怎么办？

**A:** 翻译质量取决于所使用的翻译引擎：
- 如果使用 DeepSeek API，可以尝试调整源语言设置（从 "自动检测" 改为具体语言）
- 如果使用后端 API，可以尝试重新翻译或联系支持团队

### Q: 支持哪些语言对的翻译？

**A:** 理论上支持所有 8 种语言之间的相互翻译：
- 中文 ↔ 英文、日文、韩文、西班牙文、法文、德文
- 英文 ↔ 中文、日文、韩文、西班牙文、法文、德文
- 等等

具体支持情况取决于所使用的翻译引擎。

### Q: 能否翻译多行文本？

**A:** 当前版本支持翻译单行或多行文本。只需在翻译前选中或输入要翻译的文本即可。

## 技术细节

### 翻译流程

```
用户输入文本
    ↓
选择源语言和目标语言
    ↓
点击翻译按钮
    ↓
系统检查 DeepSeek API Key
    ↓
    ├─ 有 API Key → 使用 DeepSeekDirectApi
    └─ 无 API Key → 使用 BackendAiApi
    ↓
发送翻译请求
    ↓
显示翻译结果
    ↓
用户可以提交或重新翻译
```

### 状态管理

翻译状态通过 `AiImeCoordinator` 管理：
- `translateSourceLanguage` - 当前源语言
- `translateTargetLanguage` - 当前目标语言
- `streamingPreview` - 翻译结果预览
- `isLoading` - 翻译进行中标志
- `errorMessage` - 错误信息

### 持久化存储

语言选择保存在 SharedPreferences：
- `com.arche.threply.ime.translate.sourceLanguage`
- `com.arche.threply.ime.translate.targetLanguage`
