# Threply 键盘 AI 功能使用指南

## 概述

Threply 键盘内置 AI 面板，提供三种模式：

- **B 模式**：智能回复建议（3 张回复卡片，支持长按展开子回复树）
- **C 模式**：风格控制面板（2D 拖拽调节回复长度与温度）
- **翻译模式**：文本翻译（支持多语言选择）

此外，候选栏还提供「翻译/替换/润色」技能快捷入口。

---

## 进入 AI 面板

键盘底部工具栏左侧有一个 **「AI」按钮**：

- 点击 → 打开 AI 面板（默认进入 B 模式，自动读取输入内容生成回复）
- 再次点击 → 关闭 AI 面板，回到普通键盘
- AI 激活时按钮高亮显示

AI 面板打开后，顶部显示模式切换栏：

```
[ B ] [ C ] [ 翻译 ]          [ 刷新 ] [ 扫描 ] [ 退出 ]
```

---

## B 模式（智能回复）

### 基本用法

1. 在输入框中输入一些文字
2. 点击底栏 **AI** 按钮
3. 系统自动读取输入框内容，生成 3 条回复建议
4. **点击**任意卡片 → 将该回复直接输入到输入框

### 回复树（长按展开）

1. **长按**某张回复卡片（卡片右上角有小圆点提示）
2. 系统基于该回复生成 3 条子回复
3. 顶部出现「← 返回」按钮，可回到上一层
4. 可多层展开，形成回复树

### 刷新与扫描

- **刷新**：重新生成当前模式的建议
- **扫描**：通过辅助功能读取当前聊天内容，生成更精准的回复

---

## C 模式（风格控制）

1. 点击顶部 **C** 标签切换到 C 模式
2. 出现 2D 拖拽面板：
   - **横轴**：回复长度（左短右长）
   - **纵轴**：语气温度（上温暖下克制）
3. 拖动圆点到目标位置，松手后自动保存风格偏好
4. 切回 B 模式后，新生成的回复会应用该风格

---

## 翻译模式

1. 点击顶部 **翻译** 标签
2. 选择源语言和目标语言
3. 点击翻译按钮，结果显示在面板中
4. 点击结果可直接输入到输入框

---

## 技能栏（翻译/替换/润色）

候选词栏左侧有一个 **「/」按钮**：

1. 点击展开技能栏，显示三张卡片：翻译、替换、润色
2. 点击对应卡片，对输入框中的文本执行操作
3. 替换/润色操作后会出现「撤销」按钮，支持一步还原

> 选中输入框中的文本时，技能栏会自动弹出。

---

## 退出 AI 面板

以下任一方式均可退出：

- 点击底栏 **AI** 按钮（切换关闭）
- 点击 AI 面板顶部的 **退出** 按钮

| C | AI 面板顶部 | 切换到风格控制模式 |

## 常见问题

**Q: 点击 AI 按钮没有反应？**
A: 检查是否配置了 DeepSeek API Key 或后端 API 连接。

**Q: 为什么点击"刷新"没有回复？**
A: 输入框中需要有文本内容，B 模式会读取输入框文字来生成回复。

**Q: 长按卡片没有反应？**
A: 需要按住 500ms 以上，且当前不在加载状态。

**Q: 如何返回上一层回复树？**
A: 点击面板顶部的「← 返回」按钮。
